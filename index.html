쇼핑몰 제작

Les Basics

React로 작업한 쇼핑몰 사이트.
상품을 누르면 상세페이지로 이동합니다.
장바구니에 담고, 수량을 조절할 수 있습니다.
드롭다운 메뉴에서 고객센터, 이용약관,
Our Story를 확인할 수 있습니다.


사용 기술 및 라이브러리
React, Bootstrap, Styled component, Redux, Javascript, CSS

이미지 출처
https://lesbasics.store/

Detail Page

Detail페이지는 메인페이지와의 데이터 바인딩을 위해
props를 이용해 전달해주었습니다.

각 페이지는 URL 파라미터로 연결하고,
useParams()으로 가져오도록 했습니다.

장바구니 버튼을 누르면 state.push 함수를
설정하여 추가가 되도록 했습니다.


Cart Page

Cart의 경우 Redux를 사용하여 state를
공유하도록 했습니다.

findIndex() 함수로 상품 고유의 id를 찾아,
각각 변경이 되도록 연결해주었습니다.

상품 수량을 변경할 수 있게 함수에
+, -, 삭제 파라미터 문법을 사용하였습니다.

Netflix Clone

넷플릭스 클론 코딩 '유플릭스'입니다.
각 영상을 누르면 예고편을 감상할 수 있습니다.
상단 Nav에는 홈페이지로 갈 수 있는

사용 기술 및 라이브러리
React, Bootstrap, Styled component, Firebase, Javascript, CSS, Context API, Axios, Modal, YouTube, movieTrailer

Firebase

Firebase를 이용해 백엔드를 대신하였습니다.

Row

이미지를 클릭 할 때마다 고유의 id를 가져와
movieTrailer와 연결하였습니다.

아코디언 메뉴

다시 클릭시 닫히는 토글 기능을 추가해

다크 모드

다크모드는 theme를 바꿀 수 있도록 버튼을 만들고,
default 밝은 상태에서 클릭시 다크모드로 전환됩니다.
darkTheme이 class에 추가될 경우,
다크모드로 전환되도록
darkTheme의 CSS를 추가했습니다.

localStorage에 저장됩니다.

딸기수확 게임 제작
HTML/CSS/Javascript

딸기수확게임

Javascript로 만든 딸기 수확 게임입니다.
시작버튼을 누르면 시간이 줄어들며
거미를 피해서 딸기를 클릭해야합니다.



사용 기술 및 라이브러리
HTML, CSS, Javascript

랜덤 좌표

class를 구분해 모듈 형태로 제작하였습니다.

딸기와 거미의 갯수, 이미지 경로를 인자값에 할당시켜
지정한 갯수만큼 html, style값을 생성하는 함수를 만들고,
랜덤함수를 사용하여 거미와 딸기가 윈도우 내에
임의의 좌표로 출력되도록 구현하였습니다.




타이머

setInterval()을 사용해 시간을 줄여나가고
0초가 되거나 게임에서 지면,
clearInterval()이 실행되도록 했습니다.
남은 시간이 분, 초로 표시되며,
리셋시 모든 텍스트가 초기화됩니다.


가위바위보 게임 제작
HTML/CSS/Javascript

가위바위보

Javascript로 만든 가위바위보 게임입니다.
시작버튼을 누르면 가위바위보 버튼이 활성화됩니다.
총 10판의 게임이 진행되며 컴퓨터와 대결해야합니다.
10판후 점수 및 승부 결과가 화면에 나타나고,
시작버튼을 눌러 다시 시작 할 수 있습니다.


사용 기술 및 라이브러리
HTML, CSS, Javascript

승부 결정 구조

가위바위보 각 버튼에 onclick 이벤트를 설정하고,
버튼을 누를때마다 0, 1, 2의 input 값을 받습니다.
컴퓨터는 0, 1, 2 중 랜덤한 숫자를 생성합니다.
그 값을 input 값과 비교하여 승부를 결정합니다.


결과 표시하기

9번째까지 count가 더해지고,
10번째판이 되면 win, lose, draw의 값을
비교해 최종 결과가 나타납니다.
